# General API Setup Guide

  • 1. Daftar / Login Akun
    Masuk ke website layanan (OpenAI, Google, atau BDN Portal).
  • 2. Masuk ke Dashboard / Developer Console
    Cari menu: “Developer”, “API Settings”, atau “Integrations”.
  • 3. Pilih & Aktifkan Service API
    Pilih API yang dibutuhkan.
    - Chat API: Disabled
    - Maps API: Disabled
    - Payment API: Enabled (True)
  • 4. Atur Permission (Scope)
    Tentukan akses: Read-only atau Full Access. Sangat penting untuk keamanan environment produksi.
  • 5. Simpan di Environment Variable
    Jangan hardcode key di aplikasi. Gunakan file .env atau Secret Manager.

  const response = await BDNPay.openPayment({
    amount: 50000,
    itemName: "Pro Plan Bundle"

    alert("Payment Received! ID: " + response.trxId);
  }
}
