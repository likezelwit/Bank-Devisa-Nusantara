# General API Setup Guide

  • 1. Registrasi Layanan
    Pastikan Anda sudah memiliki akun di portal Bank Devisa Nusantara sebelum mengaktifkan fitur gateway.
  • 2. Masuk ke Dashboard
    Akses Console ini untuk mengelola kunci integrasi setiap proyek Anda.
  • 3. Aktivasi Payment API
    Status layanan saat ini:
    - Chat & AI API: Disabled
    - Payment Gateway: Enabled (True)
  • 4. Keamanan Environment
    Selalu gunakan file .env untuk menyimpan API Key di server-side agar tidak terlihat oleh publik.
